# 📊 Sales Data Analysis using SQL and Excel

## 🔹 Overview  
This project focuses on analyzing sales data using SQL and Excel to extract meaningful business insights. It helps understand sales performance, profitability, and key contributing factors across regions, categories, and products.

---

## 🔹 Objectives  
- Analyze total sales and profit  
- Identify top-performing regions and categories  
- Determine best-selling products  
- Evaluate overall business profitability  
- Visualize data using an Excel dashboard  

---

## 🔹 Tools Used  
- SQL (Data Analysis)  
- Microsoft Excel (Dashboard & Visualization)

---

## 🔹 Key Analysis  
- Calculated total sales and total profit  
- Computed profit percentage (12.63%)  
- Analyzed sales by region  
- Evaluated profit by category  
- Identified top-performing products  
- Examined monthly sales trends  

---

## 🔹 Key Insights  
- West region generated the highest sales, indicating strong performance  
- Technology category produced the highest profit  
- Laptop, Sofa, and Phone are top-selling products  
- Profit margin of 12.63% indicates moderate profitability  
- Sales are concentrated among a few high-performing products  

---

## 🔹 Features  
- SQL queries for data extraction and analysis  
- Excel dashboard with pivot tables and charts  
- Visual representation of key business metrics  

---

## 📎 Files Included  
- `sales_data.csv` → Dataset used for analysis  
- `sales_dashboard.xlsx` → Excel dashboard with charts  

---

## 🚀 Conclusion  
This project demonstrates the use of SQL for data analysis and Excel for visualization to generate actionable business insights.
